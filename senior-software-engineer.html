Senior Software Engineer

Engineering Remote (Israel TZ preferred) Full-time

About Oct7

Oct7 is building the technology backbone for Jewish advocacy and community coordination. Our platform connects advocates, organizations, and communities to respond effectively to antisemitism and support Jewish causes worldwide.

The Role

We're looking for a Senior Software Engineer to join our small, high-impact engineering team. You'll work across our entire stack - from mobile apps to AI agents to infrastructure. This is a unique opportunity to shape the technical direction of a mission-driven product.

What You'll Do

  • Build and maintain our React Native mobile applications
  • Develop AI-powered features using LangGraph and LLM integrations
  • Design and implement backend services with Node.js/Python
  • Work with Neo4j graph database for relationship mapping
  • Collaborate directly with product and design on feature development
  • Contribute to architectural decisions and technical strategy

What We're Looking For

  • 5+ years of software engineering experience
  • Strong proficiency in TypeScript and Python
  • Experience with React Native or React
  • Familiarity with AI/ML concepts and LLM integrations
  • Comfortable working across the full stack
  • Self-directed and able to thrive in a small team environment
  • Passion for our mission

Nice to Have

  • Experience with graph databases (Neo4j)
  • Background in developer tools or infrastructure
  • Open source contributions
  • Experience at early-stage startups

Tech Stack

  • Frontend: React Native, Next.js, TypeScript
  • Backend: Node.js, Python, FastAPI
  • AI: LangGraph, OpenAI, Anthropic
  • Database: Neo4j, PostgreSQL
  • Infrastructure: AWS, Vercel
